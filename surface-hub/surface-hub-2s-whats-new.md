# <a name="whats-new-in-surface-hub-2s-for-it-admins"></a>Novidades no Surface Hub 2S para administradores de IT

O Surface Hub 2S é uma tela de colaboração all-in-one criada para trabalho em equipe, fornecendo um quadro de trabalho digital, uma plataforma de reuniões e uma experiência de computação do Windows 10 em um dispositivo.

| Funcionalidade | Resumo | Saiba mais |
|:-------|:------|:----|
|**Migração para Windows 10 Pro ou Enterprise**| O Surface Hub 2S vem pré-instalado com o Windows 10 Team, uma edição personalizada do Windows 10 projetada para facilitar a colaboração fácil em ambientes de sala de reunião. Agora você tem a opção de executar o Windows 10 Pro ou Enterprise para usar o Surface Hub 2S da mesma forma que qualquer outro computador.| [Anunciamos a disponibilidade do Windows 10 Pro e do Enterprise no Surface Hub 2](https://techcommunity.microsoft.com/t5/surface-it-pro-blog/announcing-the-availability-of-windows-10-pro-and-enterprise-on/ba-p/1624107) <br> <br> [Migrar para o Windows 10 Pro ou para o Enterprise no Surface Hub 2](surface-hub-2s-migrate-os.md) |
